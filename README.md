## Introducción

Este proyecto es la prueba practica para el ingreso a Weelo como desarrolladro en react native, el autor es Juan Pablo Lopez Atristizabal. La prueba consiste en que una gran empresa de bienes raíces requiere la creación de una aplicación móvil mediante la obtención de información sobre la moneda criptográfica en USD, porque la gobernanza decidió utilizar la moneda por defecto en EEUU. este necesita crear una lista de moneda criptográfica, mostrar el cambio a USD, filtrar la lista y mostrar detalles

## Intalación

#### Programas 

- Git
- Visual Studio Code
- NodeJS

#### Programas desarrollo

- React Native
- TypeScript
- Expo

#### Descarga zip

- Descargar el proyecto zip
- Abrir el proyecto en Visual Studio Code o la plataforma que utilices
- Abrir una nueva terminal y ejecutar el comando  `$ npm install` para obtner las dependencias del packge.json
- Ejecutar el comnado `$ expo start` o `$ npm run start` para iniciar el proyecto en el dispositivo de su preferencia

#### Descarga del proyecto con git

- Posicionar sobre la ruta que desea instalar el proyecto
- Abrir la consola de git en la ruta de descarga
- Ejecutar el comando `$ git clone 'ruta del proyecto en git'` para instarlo localmente
- Abrir el proyecto en Visual Studio Code o la plataforma que utilices
- Abrir una nueva terminal y ejecutar el comando  `$ npm install` para obtner las dependencias del packge.json
- Ejecutar el comnado `$ expo start` o `$ npm run start` para iniciar el proyecto en el dispositivo de su preferencia

## Estrutura del proyecto

La carpeta principal del proyecto es src, donde todo el desarrollo del proyecto

- **components:** En esta carpeta se encuentran los componenetes generales del proyecto
- **model:** En este se encuentran las interfaces del proyecto
- **screen:** Aqui estan las vistas de usuario
- **service:** En este directorio estan los servicios a las petciones web de API de CoinLore
- **styles:** Hojas de estilo para las vistas de usuario y componenetes
- **utils:** Desarrollo general del proyecto

**Nota:** En la carpeta assets se encuentran las fuentes de la letra y las imagenes e iconos del proyecto, pero en este caso no los modifique solo lasm fuentes se pusieron unos ttf para el tipo de letra
